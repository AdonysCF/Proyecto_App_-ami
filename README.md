🍔 Proyecto Ñami: Arquitectura de Datos para App de Delivery
📌 Descripción
Proyecto académico que diseña la arquitectura de datos para una aplicación de delivery, abarcando desde sistemas transaccionales hasta analítica y pruebas de escalabilidad.

🛠️ Tecnologías Utilizadas
OLTP: SQL Server (Procedimientos almacenados, Triggers)

OLAP (Data Warehouse): SQL Server (Modelo Estrella)

Business Intelligence: Power BI

Big Data: Apache Spark (PySpark) en Google Colab

NoSQL: MongoDB (Prueba de Concepto)

🚀 Cómo probar este proyecto
Ejecutar los scripts de la carpeta 1_Base_De_Datos_SQL en SQL Server.

El archivo de Power BI se conecta al DW de SQL Server.

El cuaderno de Spark se puede abrir directamente en Google Colab importando el archivo .ipynb.
